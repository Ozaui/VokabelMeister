A-05 — Sistem Kelimesi API (Words) (tamamlandı)

Uygulamanın asıl içeriği: Almanca-Türkçe kelime kartları. Çoklu dil desteği tek bir "Kelime" tablosuyla değil, dilden bağımsız bir WordConcept (tür/seviye/görsel) + her dile bir Word satırı modeliyle çözülüyor. Entity + EF Config + Migration + Language seed (de/tr) + WordGrammarValidator + Repository + 5 Command/Query (CRUD) + Controller + testler, ardından "Eşleştirme" akışı (GetUnmatchedWordConceptsQuery + PairWordConceptsCommand — ayrı ayrı import edilen de/tr içeriğini birleştirme) işlendi.

  1. 1 Neden WordConcept? Tek tablo yerine kavram+dil-başına-satır modeli, BaseEntity var/yok kararı.
  2. 2 Entity + EF Config Language.cs (BaseEntity yok) + WordConcept.cs + CHECK constraint'ler; Word/WordDetail/WordExample aynı desen.
  3. 3 Migration + Seed + DbSet'ler AddWordsSchema — 5 tablo tek migration'da, Languages seed (de/tr), DbContext'e 5 yeni DbSet.
  4. 4 WordGrammarValidator — Dile ve Türe Göre Dallanan Doğrulama AbstractValidator<WordGrammarInput>, System.Text.Json ile alan kontrolü, DE/TR Noun/Verb matrisleri, 22 ErrorMessages kodu, 23 birim testi.
  5. 5 Repository Katmanı — WordConcept + Language Aggregate root deseni, IWordConceptRepository (liste/detay/duplikat/cascade soft delete), ILanguageRepository, DuplicateWordException.
  6. 6 Command/Query Handler'ları — Create/Update/Delete/Get WordDtos, WordEntityBuilder/WordConceptDtoBuilder, 5 Handler + Postman örnekleri (force/eşleştirme dahil).
  7. 7 Controller + Validator Entegrasyonu + Handler Testleri CreateWordCommandValidator (WordGrammarValidator entegrasyonu), WordsController (ilk Admin rolü), 17 birim testi.
  8. 8 Eşleştirme — Repository + Query/Command WordMatchSuggestionResolver (iki yönlü Definition↔Text öneri), PairAsync, GetUnmatchedWordConceptsQuery, PairWordConceptsCommand + Validator.
  9. 9 Eşleştirme — Controller + Postman + Testler GET /words/unmatched, POST /words/pair, Postman örnekleri, "bloklayıcı hata yok" testleri, 9 yeni birim testi.
  10. 10 Özet + Sözlük A-05'in tamamı: 5 entity, 7 Command/Query, 7 endpoint, 193/193 yeşil test.
