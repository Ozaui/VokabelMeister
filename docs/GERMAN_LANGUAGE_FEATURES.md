# Kelime Öğrenme Uygulaması - Almanca Spesifik İçerik

## 1. Almanca Dil Yapısı

### 1.1 Temel Özellikler

Almanca, İngilizceye göre daha karmaşık bir yapıya sahiptir. Başlıca zorluklar:

```
1. Gender (Cinsiyetlik)      → 3 cinsiyet (der, die, das)
2. Articles (Artikeller)      → Belirsiz ve Belirli artikeller
3. Cases (Haller)             → 4 farklı hâl (Nominatif, Akkusatif, Dativ, Genitiv)
4. Plural Forms (Çoğullar)    → Değişken kurallar
5. Verb Conjugation (Çekimi)  → Kişiye göre değişim
6. Word Order (Sözcük Sırası) → Farklı cümle tiplerine göre
7. Separable Verbs (Ayrılabilir Fiiller) → Ön ek ayrılması
8. Compound Words (Bileşik Kelimeler) → Çok uzun kelimeler
```

### 1.2 Cinsiyet (Gender) Sistemi

```
┌──────────────────────────────────────────────┐

│ ✓ Kişiler (Menschen, Männer)                 │
```

### 1.3 Artikeller Tablosu

#### 1.3.1 Belirli Artikeller (Definite Articles)
```
         NOMINATIF  AKKUSATIF  DATIV  GENITIV
MASKULIN    der        den       dem    des
FEMİNİN     die        die       der    der
NEUTRİ      das        das       dem    des
ÇOĞUL       die        die       den    der

Örnek:
┌─────────────────────────────────────────┐
│ der Mann (erkek) - Maskulin             │
├──────────────┬──────────────┬───────────┤
│ Nominativ    │ Der Mann     │ Özne      │
│ Akkusativ    │ den Mann     │ Nesne     │
│ Dativ        │ dem Mann     │ Dolaylı   │
│ Genitiv      │ des Mannes   │ İyelik    │
└──────────────┴──────────────┴───────────┘

Cümle örnekleri:
• Der Mann arbeitet. (Erkek çalışıyor)           [Nominativ]
• Ich sehe den Mann. (Erkek görüyorum)           [Akkusativ]
• Ich helfe dem Mann. (Erkeğe yardım ediyorum)   [Dativ]
• Das Auto des Mannes. (Erkeğin arabası)         [Genitiv]
```

#### 1.3.2 Belirsiz Artikeller (Indefinite Articles)
```
         NOMINATIF  AKKUSATIF  DATIV  GENITIV
MASKULIN    ein        einen      einem   eines
FEMİNİN     eine       eine       einer   einer
NEUTRİ      ein        ein        einem   eines
ÇOĞUL       -          -          -       -
            (kein, keine, kein, keine - olumsuzda)

Örnek:
• Ein Mann arbeitet. (Bir erkek çalışıyor)
• Ich sehe einen Mann. (Bir erkek görüyorum)
• Ich helfe einem Mann. (Bir erkeğe yardım ediyorum)
• Das Auto eines Mannes. (Bir erkeğin arabası)

Olumsuzda (Negation):
• kein Mann (hiçbir erkek)
• keine Frau (hiçbir kadın)
• kein Kind (hiçbir çocuk)
• keine Kinder (hiçbir çocuk - çoğul)
```

## 3. Dört Hâl Sistemi (Cases)

### 3.1 Hâller Tanımı

```
┌───────────────────────────────────────────────────┐
│ NOMINATIF (Nominative) - Özne Hâli                │
├───────────────────────────────────────────────────┤
│ Soru: Wer? Was? (Kim? Ne?)                        │
│ Fonksiyon: Cümlenin özne'sidir                    │
│                                                   │
│ Örnekler:                                         │
│ • Der Mann geht. (Erkek gidiyor)                  │
│ • Die Frau sitzt. (Kadın oturuyor)                │
│ • Das Kind spielt. (Çocuk oynuyor)                │
│ • Die Lehrer arbeiten. (Öğretmenler çalışıyor)   │
└───────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────┐
│ AKKUSATIF (Accusative) - Nesne Hâli               │
├───────────────────────────────────────────────────┤
│ Soru: Wen? Was? (Kimi? Neyi?)                     │
│ Fonksiyon: Doğru nesne'dir                        │
│                                                   │
│ Örnekler:                                         │
│ • Ich sehe den Mann. (Erkek görüyorum)            │
│ • Sie küsst die Frau. (Kadını öpüyor)             │
│ • Er isst das Brot. (Ekmeği yiyor)                │
│ • Wir lieben die Kinder. (Çocukları seviyoruz)    │
└───────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────┐
│ DATIV (Dative) - Dolaylı Nesne Hâli               │
├───────────────────────────────────────────────────┤
│ Soru: Wem? Wo? (Kime? Nereye?)                    │
│ Fonksiyon: Dolaylı nesne'dir                      │
│                                                   │
│ Örnekler:                                         │
│ • Ich helfe dem Mann. (Erkeğe yardım ediyorum)    │
│ • Sie zeigt der Frau. (Kadına gösteriyor)         │
│ • Ich gebe dem Kind ein Buch. (Çocuğa kitap verir)│
│ • Er dankt den Lehrern. (Öğretmenlere teşekkür)   │
└───────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────┐
│ GENITIV (Genitive) - İyelik Hâli                  │
├───────────────────────────────────────────────────┤
│ Soru: Wessen? (Kimin?)                            │
│ Fonksiyon: İyeliği ifade eder                     │
│                                                   │
│ Örnekler:                                         │
│ • Das Auto des Mannes. (Erkeğin arabası)          │
│ • Die Tasche der Frau. (Kadının çantası)          │
│ • Das Spielzeug des Kindes. (Çocuğun oyuncağı)    │
│ • Die Bücher der Lehrer. (Öğretmenlerin kitapları)│
│                                                   │
│ Modern kullanım (Dativ'i yoğun şekilde yeğlenir):│
│ • Die Tasche von der Frau. (Kadının çantası)      │
└───────────────────────────────────────────────────┘
```

## 4. Ayrılabilir Fiiller (Separable Verbs)

```
Ayrılabilir fiiller Almancaya has bir durumdur.
Ön ek ana fiilden ayrılır ve cümlenin sonuna gider.

┌──────────────────────────────────────────────┐
│ ÖRNEKLER                                     │
├──────────────────────────────────────────────┤
│ anrufen (aramak) → an + rufen                │
│ • Ich rufe dich an. (Seni arıyorum)          │
│ • Er hat mich angerufen. (Beni aradı)        │
│                                               │
│ aufstehen (kalkmak) → auf + stehen           │
│ • Ich stehe um 6 Uhr auf. (6'da kalkıyorum)  │
│ • Sie ist früh aufgestanden. (Erken kalkı)   │
│                                               │
│ einschlafen (uyanmak) → ein + schlafen       │
│ • Das Kind schläft schnell ein. (Çocuk çabuk │
│   uyuyor)                                     │
│                                               │
│ fernsehen (TV izlemek) → fern + sehen        │
│ • Ich sehe jeden Abend fern. (Her gece TV    │
│   izlerim)                                    │
│                                               │
│ mitkommen (gelmek) → mit + kommen            │
│ • Komm mit! (Gel beraber!)                   │
│                                               │
│ weggehen (gitmek) → weg + gehen              │
│ • Ich gehe jetzt weg. (Şimdi gidiyorum)      │
└──────────────────────────────────────────────┘

Yaygın Ön Ekler:
ab- (ab)        aus- (dış)      anrufen (ara)
auf- (açık)     ein- (içeri)    aus- (dış)
fest- (sıkı)    los- (serbest)  mit- (birlikte)
um- (etrafında)  vor- (önce)    weg- (uzak)
zu- (kapalı)    zurück- (geri)  zusammen- (birlikte)
```

## 5. Bileşik Kelimeler (Compound Words)

```
Almanca karakteristik özellik: Kelimeler birlev yolu ile oluşturulur.

AÇIK TIP BİLEŞİK:
┌─────────────────────────────────────────────────┐
│ Fahrrad (bisiklet)                              │
│ = fahren (gitmek) + Rad (tekerlek)              │
│                                                  │
│ Tischlampe (masa lambası)                       │
│ = Tisch (masa) + Lampe (lamba)                  │
│                                                  │
│ Schwarzwalt (Schwarzwald - siyah orman)         │
│ = schwarz (siyah) + Wald (orman)                │
│                                                  │
│ Zahnbürste (diş fırçası)                        │
│ = Zahn (diş) + Bürste (fırça)                   │
│                                                  │
│ Lieblingsmusik (favori müzik)                   │
│ = Liebling (favori) + Musik (müzik)             │
│                                                  │
│ Geschäftszimmer (ofis odası)                    │
│ = Geschäft (iş) + Zimmer (oda)                  │
│                                                  │
│ Küchentisch (mutfak masası)                     │
│ = Küche (mutfak) + Tisch (masa)                 │
│                                                  │
│ Freundschaftlichkeit (dostluk)                  │
│ = Freund (arkadaş) + Schaft + lichkeit + heit  │
└─────────────────────────────────────────────────┘

Cinsiyet Kuralı:
Bileşik kelimenin cinsiyeti, son kelimenin cinsiyetine dayanır.
• der Apfel (elma) + der Baum (ağaç) = der Apfelbaum
• die Küche (mutfak) + der Tisch (masa) = der Küchentisch
• das Haus (ev) + die Tür (kapı) = die Haustür
```

## 6. Entity Structure (Database)

### 6.1 WordDetail Entity
```csharp
public class WordDetail
{
    public int Id { get; set; }
    public int WordId { get; set; }
    
    // Cinsiyetlik
    public GenderType Gender { get; set; } // enum: Masculine, Feminine, Neuter
    
    // Belirli Artikeller
    public string ArticleDefinite { get; set; } // der, die, das
    public string ArticleDefiniteAcc { get; set; } // den, die, das
    public string ArticleDefiniteDat { get; set; } // dem, der, dem
    public string ArticleDefiniteGen { get; set; } // des, der, des
    
    // Belirsiz Artikeller
    public string ArticleIndefinite { get; set; } // ein, eine, ein
    public string ArticleIndefiniteAcc { get; set; } // einen, eine, ein
    public string ArticleIndefiniteDat { get; set; } // einem, einer, einem
    public string ArticleIndefiniteGen { get; set; } // eines, einer, eines
    
    // Tekil Formlar (4 Case)
    public string Nominative { get; set; } // Örnek: Mann
    public string Accusative { get; set; } // Örnek: Mann (aynı)
    public string Dative { get; set; } // Örnek: Mann (aynı)
    public string Genitive { get; set; } // Örnek: Mannes
    
    // Çoğul Formlar
    public string PluralForm { get; set; } // Örnek: Männer
    public string PluralArticle { get; set; } // Daima "die"
    public string PluralNominative { get; set; }
    public string PluralAccusative { get; set; }
    public string PluralDative { get; set; } // Nominative + -n (Männern)
    public string PluralGenitive { get; set; }
    
    // Fiiller İçin
    public string PresenceForm { get; set; } // Ben: gehe, Du: gehst, Er/Sie/Es: geht
    public bool IsSeparableVerb { get; set; } // ayrılabilir fiil mi?
    public string SeparablePrefix { get; set; } // Örnek: "an" (anrufen)
    public string ConjugationJson { get; set; } // JSON: ich/du/er-sie-es/wir/ihr/sie conjugations
    
    // Genel
    public string ExampleSentenceDE { get; set; }
    public string ExampleSentenceTR { get; set; }
    public string Pronunciation { get; set; } // IPA notation
    public string ImageUrl { get; set; }
    public string AudioUrl { get; set; }
    public string Notes { get; set; }
    
    public DateTime CreatedAt { get; set; }
    public DateTime UpdatedAt { get; set; }
    
    // Navigation
    public virtual Word Word { get; set; }
}

// Enum tanımları
public enum GenderType
{
    Masculine,    // der
    Feminine,     // die
    Neuter        // das
}

public enum CaseType
{
    Nominative,   // Wer/Was?
    Accusative,   // Wen/Was?
    Dative,       // Wem/Wo?
    Genitive      // Wessen?
}
```

### 6.2 İlişkili Service
```csharp
public interface IWordGermanService
{
    // Cinsiyeti güncelle
    Task UpdateGenderAsync(int wordId, GenderType gender);
    
    // Hâl formlarını güncelle
    Task UpdateCaseFormsAsync(int wordId, CaseForms forms);
    
    // Çoğul formunu güncelle
    Task UpdatePluralAsync(int wordId, PluralForm plural);
    
    // Ayrılabilir fiil olarak işaretle
    Task MarkAsSeparableVerbAsync(int wordId, string prefix);
    
    // Fiil çekimlerini kaydet
    Task SaveConjugationsAsync(int wordId, ConjugationData conjugations);
    
    // Tüm formları getir (kart gösterimi için)
    Task<WordGermanDetailDto> GetCompletFormsAsync(int wordId);
}

// DTO Sınıfları
public class CaseForms
{
    public string Nominative { get; set; }
    public string Accusative { get; set; }
    public string Dative { get; set; }
    public string Genitive { get; set; }
}

public class PluralForm
{
    public string Form { get; set; }
    public string Nominative { get; set; }
    public string Accusative { get; set; }
    public string Dative { get; set; }
    public string Genitive { get; set; }
}

public class ConjugationData
{
    public string Ich { get; set; }     // I
    public string Du { get; set; }      // You (singular)

    public string Wir { get; set; }     // We
    public string Ihr { get; set; }     // You (plural)
    public string Sie { get; set; }     // They/You formal
}
```

## 8. Learning Path Strukturu

### 8.1 Seviye Sistemi

```
A1 (Anfänger - Başlangıç):
├─ Temel saatler (Stunden)
├─ Temel fiiller (gehen, sein, haben, machen)
├─ Basit isimler (Mann, Frau, Kind)
├─ Cinsiyetlik tanıma
├─ Basit çoğullar
└─ Günlük kalıplar

A2 (Elementar - Temel):
├─ Daha fazla fiil (essen, trinken, schlafen)
├─ Sıfatlar (schön, groß, klein)
├─ Dative durumu tanıtımı
├─ Ayrılabilir fiiller
├─ Geçmiş zaman (Präteritum, Perfekt)
└─ Hocası ile konuşmalar

B1 (Mittelstufe - Orta):
├─ Karmaşık fiiller
├─ Tüm 4 hâl sistemi
├─ Bileşik kelimeleri anlama
├─ Konjunktiv
├─ Zamanlar: Futur, Plusquamperfekt
└─ Yazılı iletişim

B2 (Oberstufe - İleri):
├─ İdiyomatik ifadeler
├─ Karmaşık yapılar
├─ Hukuki/resmi Almanca
├─ Ön ekler derin analiz
└─ Yazı vs konuşma farkları

C1 (Fortgeschrittene):
├─ Teknik ve akademik metinler
├─ Stilistik incelikler
└─ Kültürel referanslar

C2 (Mastery):
├─ Edebiyat analizi
├─ Diyalektler
└─ Tam yeterlilik
```
